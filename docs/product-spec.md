# 제품 명세서

## 제품 개요

일본어 원서와 웹소설 학습자가 원문을 붙여넣으면, 학습 가치가 있는 단어만 추출해 한국어 뜻과 함께 확인하고 개인 단어장으로 저장할 수 있는 웹서비스를 만든다.

MVP는 작고 안정적인 단어 추출, 분류, 저장, CSV 내보내기에 집중한다. AI 문맥 설명, 사용자 계정, 고급 사전 기능은 추후 단계로 미룬다.

## 기본 저장 정책

- 사용자가 붙여넣은 일본어 원문 전체는 서버 DB에 저장하지 않는다.
- 원문은 분석 요청 처리 중에만 사용하고, 응답 생성 후 폐기한다.
- 저작권 이슈를 줄이기 위해 저장 대상은 사용자가 선택한 단어장 데이터로 제한한다.
- 저장되는 데이터는 단어의 표면형, 기본형, 읽기, 품사, 한국어 뜻, 사용자의 학습 상태 등 단어장 운영에 필요한 최소 정보로 제한한다.

## MVP 기능

### 원문 붙여넣기

- 사용자는 일본어 원문을 텍스트 입력창에 붙여넣는다.
- 입력한 원문을 분석 요청으로 서버에 전송한다.
- MVP에서는 파일 업로드, URL 크롤링, 브라우저 확장 프로그램 입력은 지원하지 않는다.

### 형태소 분석과 단어 추출

- 서버는 SudachiPy로 일본어 형태소 분석을 수행한다.
- 분석 결과에서 표면형, 기본형, 읽기, 품사를 추출한다.
- 조사, 조동사, 기호 등 학습 가치가 낮은 토큰은 제외한다.
- 기본형 기준으로 중복 단어를 제거한다.
- MVP에서는 단어 빈도, 문장별 출현 위치, 난이도 추정은 필수 기능으로 포함하지 않는다.

### 한국어 뜻 표시

- 분석 결과 단어마다 한국어 뜻을 표시한다.
- MVP의 뜻 데이터는 로컬 사전, 고정 매핑, 또는 서버 내부 사전 테이블을 우선 사용한다.
- 뜻을 찾지 못한 단어는 빈 값 또는 "뜻 미등록" 상태로 표시할 수 있다.
- AI 번역이나 문맥 기반 뜻 생성은 MVP에서 제외한다.

### 아는 단어 / 모르는 단어 분류

- 사용자는 분석 결과 목록에서 각 단어를 아는 단어 또는 모르는 단어로 분류한다.
- 기본 상태는 미분류로 둘 수 있다.
- 모르는 단어로 선택한 항목만 단어장 저장 대상으로 삼는다.

### 단어장 저장

- 사용자는 모르는 단어를 단어장에 저장한다.
- 저장 시 원문 전체나 문단 전체는 저장하지 않는다.
- 동일 기본형의 단어가 이미 저장되어 있으면 중복 저장하지 않고 기존 항목을 유지하거나 갱신한다.

### CSV 내보내기

- 저장된 단어장을 CSV 파일로 내보낸다.
- CSV에는 표면형, 기본형, 읽기, 품사, 한국어 뜻, 학습 상태, 저장일을 포함한다.

## MVP 제외 기능

- AI 문맥 설명
- AI 번역 또는 AI 기반 뜻 생성
- 원문 전체 저장
- 사용자 계정과 로그인
- 여러 단어장 관리
- 원문 파일 업로드
- 웹소설 URL 자동 수집 또는 크롤링
- 브라우저 확장 프로그램
- SRS 복습 알고리즘
- 예문 저장
- 단어 빈도 분석 고도화
- 품사 필터 사용자 설정
- 모바일 앱

## 사용자 흐름

1. 사용자가 웹페이지에 접속한다.
2. 일본어 원문을 입력창에 붙여넣는다.
3. "분석" 버튼을 누른다.
4. 서버가 원문을 형태소 분석하고 학습 후보 단어 목록을 반환한다.
5. 사용자는 단어의 표면형, 기본형, 읽기, 품사, 한국어 뜻을 확인한다.
6. 사용자는 각 단어를 아는 단어 또는 모르는 단어로 분류한다.
7. 사용자는 모르는 단어를 단어장에 저장한다.
8. 사용자는 저장된 단어장을 확인한다.
9. 필요하면 단어장을 CSV로 내보낸다.

## 성공 기준

- 사용자가 붙여넣은 일본어 원문에서 학습 후보 단어를 안정적으로 추출한다.
- 조사, 조동사, 기호 등 불필요한 토큰이 기본적으로 제외된다.
- 같은 기본형의 단어가 중복으로 표시되거나 저장되지 않는다.
- 사용자가 모르는 단어를 저장하고 CSV로 받을 수 있다.
- 서버 DB에 원문 전체가 저장되지 않는다.
